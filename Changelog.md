# Changelog

All notable changes to the Accessibility Widget project will be documented in this file.

## [1.0.0] - 11-19-2025

### Added
- Initial release of Accessibility Widget
- **Text & Display Features**:
  - Font size adjustment (75%, 100%, 125%, 150%, 200%)
  - Contrast modes (Normal, High Contrast, Dark Mode)
  - Line height control (Normal, Large)
  - Letter spacing (Normal, Wide)
  - Font family options (Default, Sans Serif, Serif, Monospace)
  
- **Color & Vision Support**:
  - Protanopia color blindness filter
  - Deuteranopia color blindness filter
  - Tritanopia color blindness filter
  - Achromatopsia color blindness filter
  - High contrast mode
  
- **Reading & Navigation Tools**:
  - Text-to-speech (TTS) with adjustable rate, pitch, and volume
  - Reading guide that follows cursor
  - Reading mask to hide text below reading position
  - Text highlight feature
  - Image alt text display
  
- **Accessibility Tools**:
  - Enhanced focus indicator
  - Stop animations option
  - Underline all links
  - Full keyboard navigation support
  - Persistent settings via localStorage
  
- **Developer Features**:
  - JavaScript API for programmatic control
  - Configuration file for customization
  - CDN support via jsDelivr
  - WordPress compatibility
  - Example files (HTML & PHP)
  
- **Documentation**:
  - Comprehensive README
  - Release notes
  - API documentation
  - Installation guides
  - Example code snippets

### Technical Details
- Pure JavaScript and CSS (no dependencies)
- Self-contained plugin (no server-side configuration)
- Responsive design (desktop and mobile)
- Browser compatibility (Chrome, Firefox, Safari, Edge)
- WCAG guidelines compliance
- Performance optimized

### Browser Support
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)

### Known Issues
- Color blindness filters may have slight performance impact on very large pages
- TTS requires browser support for Web Speech API
- Some older browsers may not support all features
- Widget Sticks to the Bottom when on all of the color blindness settings and on High Contrast
